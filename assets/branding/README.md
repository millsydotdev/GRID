# GRID Branding

**Status**: No branding assets exist yet.

## What Will Go Here

When GRID's visual identity is established, this folder will contain:

- **Logo files**:
  - `logo.svg` - Primary logo (SVG for scalability)
  - `logo.png` - Primary logo (PNG variants)
  - `logo-dark.svg` / `logo-light.svg` - Theme-specific versions
  - `icon.png` - App icon (various sizes)

- **Brand guidelines**:
  - Color palette
  - Typography
  - Usage guidelines

## Current Approach

GRID currently uses text-based placeholders:
- Onboarding screen: "GRID" text in circular frame
- Editor watermark: "GRID" text placeholder
- README: No visual, text-only header

## When You Design a Logo

DO NOT use:
- Generic AI/robot imagery
- Stock graphics
- Repurposed third-party assets

DO consider:
- Uniqueness and memorability
- Scalability (works at all sizes)
- Theme compatibility (dark/light modes)
- Simplicity and clarity
