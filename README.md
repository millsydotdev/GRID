# GRID

GRID is the open-source Cursor alternative.

Use AI agents on your codebase, checkpoint and visualize changes, and bring any model or host locally. GRID sends messages directly to providers without retaining your data.

This repo contains the full sourcecode for GRID. If you're new, welcome!
